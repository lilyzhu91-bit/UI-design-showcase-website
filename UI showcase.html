01 — Context

ユーザーの本質的なニーズ

01

多通貨の可視化

世界中に分散した資産（円、ドル、元など）を一つのダッシュボードで統合管理する。

02

データ主権

資産データはローカル環境に留める。銀行や証券口座とのAPI連携を排し、トラッキングされない「クリーンな環境」を追求。

03

履歴追跡

資産配分の経時的な変化を可視化し、長期的な意思決定をサポートする。

03 — Aesthetics

データのノイズを排し、「読む」体験へ

Design Principle

静謐なインターフェース

投資アプリ特有の刺激的な配色を避け、
市場の変動に惑わされない「静謐なインターフェース」を追求。

明朝体と大胆な余白を活かし、
信頼感のある上質な誌面のようなレイアウトを採用。

文字そのものの美しさと配置によって、
情報の論理構造を視覚的に構築する。

Wireframe
Final UI
Philosophy

Less noise, more clarity. 市場の変動より長期的な視野。

Typography Approach

Every character placement constructs logical hierarchy. 活字配置が情報構造を語る。

Target User

Cross-border asset holders who value data sovereignty over cloud convenience. データ主権を重視するユーザー。

05 — Data Visualization

データ可視化の論理

二重円環の構造

Dual-Ring Chart

内側（固定分類5種）と外側（ユーザー定義）の二重構造。
同系色マッピングにより、0.5秒で資産構成の健全性を直感的に判断可能に。

  • 内側：固定5分類（現金・株式・債券・不動産・その他）
  • 外側：ユーザー定義の詳細カテゴリ
  • 同系色マッピングで階層を即時把握
二状態スタックチャート
二状態スタックチャート

Dual-State Stack Chart

シェア表示と金額表示の二状態を切り替え。リスクエクスポージャーと純資産の成長の両軸から資産を読み解く。

  • シェア表示 (%): 配分の安定性・リスクを可視化
  • 金額表示 (Amount): 純資産の蓄積と成長を可視化
  • 同系色マッピングで0.5秒判断

06 — Interaction

直感でインタラクション

多通貨視点の動的制御

Multi-Currency Control

行（Row）の動的追加・削除機能により、ポートフォリオの変更に即座に対応する。

  • 各項目ごとに独立した通貨設定が可能
  • 総資産ブロックで合計表示の通貨設定が可能
  • リアルタイムでの為替換算表示
時系列の動的分析
時系列の動的分析

Timeline Analysis

モバイル画面に最適化された、タップ＆スライドによる時系列分析インターフェース。

  • Y軸固定・X軸スライドによるモバイル最適化
  • タップ操作で特定の時点の詳細内訳を瞬時に表示
